So what's this all about?

This is a wifi hotspot that only gives you access to the Noagenda podcast. The NoAgenda show deconstructs corporate media propaganda.

Free speech is villified by coporate press and weird, conformity-minded narcissists. Big tech is becoming the access control that prevents the free exchange of ideas. This hotspot exists to combat the mechanisms that transform creative, enduring humans into compliant citizens. Please enjoy. If you find these ideas valuable, make a donation to the No Agenda Show at dvorak.org/na.
